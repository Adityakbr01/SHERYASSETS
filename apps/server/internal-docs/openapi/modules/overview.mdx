## Internal docs workflow

1. Update route/controller/validation in module source.
2. Update module OpenAPI registration in `openapi/modules/<module>`.
3. Run bun run docs:generate:internal.
4. Run bun run test:docs.
5. Preview internal docs and API Reference.

## Separation contract

- Internal docs config is internal-docs/docs.json.
- Internal specs are generated only under internal-docs/.
- Public docs config/specs remain fully independent under docs/.

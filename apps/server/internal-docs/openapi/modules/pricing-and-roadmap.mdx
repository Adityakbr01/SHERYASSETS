## Pricing tiers

| Plan          | Best for                 | Highlights                        |
| ------------- | ------------------------ | --------------------------------- |
| Basic         | Getting started          | Free onboarding, lower limits     |
| Pro           | Production teams         | Higher limits, better performance |
| Pay as you go | Variable traffic         | Usage-based flexibility           |
| Enterprise    | High-scale organizations | Custom limits, SLA, support       |

## Product direction

Current direction focuses on:

- Fast onboarding with auth and tenant defaults
- Clear billing and plan transitions
- Secure API key workflows
- Reliable webhook-driven updates

## Engineering roadmap

1. Stabilize upload and transformation pipeline
2. Expand usage metering and analytics
3. Improve enterprise governance controls
4. Harden observability and incident workflows

## Documentation roadmap

1. Keep OpenAPI examples synced with tested behaviors
2. Expand module-level troubleshooting tables
3. Add end-to-end recipes for common developer tasks

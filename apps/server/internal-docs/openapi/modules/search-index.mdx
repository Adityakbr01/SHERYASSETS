## Use built-in search

Mintlify provides built-in search across all pages in this docs site.

Tips:

- Search by endpoint fragment like switch-tenant or subscribe/verify
- Search by status code like 401, 403, 404, 429
- Search by error phrases like invalid signature or plan limit
- Search old slug alias plan-roadmap to find the renamed roadmap page

## Suggested endpoint map template

| Method | Endpoint                 | Auth           | Purpose               |
| ------ | ------------------------ | -------------- | --------------------- |
| POST   | /api/v1/upload           | API key or JWT | Upload original image |
| GET    | /api/v1/upload/{assetId} | JWT            | Fetch asset metadata  |
| DELETE | /api/v1/upload/{assetId} | JWT            | Soft delete asset     |

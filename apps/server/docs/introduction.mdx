## Welcome

SheryAssets is a backend for auth, teams, plans, billing, and API keys.
These docs use simple language and practical steps.

## First 30 minutes path

1. Open /getting-started
2. Complete /developer-onboarding-checklist
3. Run one flow from /api-recipes
4. Check request format in /api-reference
5. If error happens, open /troubleshooting

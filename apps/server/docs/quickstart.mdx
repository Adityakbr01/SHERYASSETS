## 1. Authentication

To interact with our API, you'll need a valid API key or a JWT from our Auth module. Follow the [Auth Documentation](/authentication) for more details.

## 2. Setting Up Your Module

If you're building a custom module, ensure it's placed in `src/modules` and documented in the `docs/openapi` layer.

## 3. Explore the API

Head over to our [API Reference](/api-reference) to start making your first requests. Our documentation is automatically synced with our Zod schemas, meaning it's always up-to-date.

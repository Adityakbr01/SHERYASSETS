## Welcome to SheryAssets API

The SheryAssets API provides seamless, scalable asset upload, delivery, and transformation for your applications. Our platform is built for speed, relying on global CDN infrastructure and on-demand image optimization so your users get lightning-fast media.

## Who is this for?

This documentation is for developers looking to integrate image uploading and processing directly into their products. You interact with our simple HTTP endpoints to:

- **Authenticate** to manage your workspace via your API keys.
- **Upload** raw assets and images into the platform.

## Getting Started

To get started quickly, check out the [Quickstart guide](/openapi/modules/quickstart).

## Key Concepts

- **API Keys**: Securely authenticate to the API.
- **Assets**: The files you upload (like `.png`, `.jpg`, `.webp`) that are processed and distributed globally.
- **Transformations**: Instantly crop, resize, and optimize images directly via URL parameters on the delivery system.
